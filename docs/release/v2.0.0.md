# v2.0.0 リリースノート

バックグラウンドプロセスの管理機能を強化したメジャーリリース。`mdc proc kill` コマンドにより、子プロセスツリーを含めたプロセスの強制終了が可能になった。また、Homebrew からのインストールに対応し、導入がさらに簡単になった。

## 主な機能

- **`mdc proc kill` コマンド**: バックグラウンドプロセスとその子プロセスツリーを強制終了 (#21)
- **Homebrew 対応**: `brew install tominaga-h/tap/mdc` でインストール可能に

## 内部改善

- プロセス終了処理の改善（子プロセスツリーの追跡と終了）
- README に Homebrew インストール手順を追加

## 対応プラットフォーム

- macOS / Linux (PTY 対応)
- Windows (PTY なしのフォールバック)

## 既知の制限

- `mdc ps` コマンドは `docker` / `docker compose` がインストールされた環境でのみ動作
