# v1.0.0 リリースノート

Phase 3（DX 向上）の全機能が揃い、安定版として初のメジャーリリース。設定ファイルの作成・編集・削除をコマンドラインから完結できるようになり、日常的なワークフローが大幅に改善された。

## 主な機能

- **`mdc init` コマンド**: 設定ファイルのテンプレートを新規作成。`--edit` オプションで作成後すぐにエディタで開くことも可能 (#17)
- **`mdc edit` コマンド**: 環境変数 `$EDITOR`（未設定時は `vim`）で設定ファイルを直接編集 (#19)
- **`mdc rm` コマンド**: 設定ファイルの削除。確認プロンプト付きで、`--force` オプションによるスキップにも対応 (#20)

## 内部改善

- README を英語化し、日本語版を `docs/README_ja.md` に分離 (#18)
- デモ GIF を README に追加し、ツールの動作を視覚的に紹介
- README 全体の見直し・コマンドリファレンスの充実

## 対応プラットフォーム

- macOS / Linux (PTY 対応)
- Windows (PTY なしのフォールバック)

## 既知の制限

- `mdc ps` コマンドは `docker` / `docker compose` がインストールされた環境でのみ動作
